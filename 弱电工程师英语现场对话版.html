弱电工程师英语现场对话版

适用场景：网络、桌面、打印机、综合布线、监控、门禁、现场施工、工单沟通

目标：不会复杂英语也能开口说

原则：只学工作里真会用到的句子

1. 使用方法

这份不要当语法书看。

直接当“现场话术本”用。

建议这样学：

  • 先看中文意思
  • 再大声读英文
  • 直接套用到自己的工作场景
  • 每天背 5 到 10 句

如果你英语很差，就先背短句，不要追求完整语法。

2. 最常用开场句

你到现场时

  • 你好，我来检查这个问题。
  • 我是 IT 支持。
  • 我是来处理网络问题的。
  • 我来检查电脑。
  • 我来检查布线。

你想确认问题时

  • 是什么问题？
  • 你能给我看一下问题吗？
  • 是什么时候开始的？
  • 之前能用吗？
  • 这个问题是一直都有吗？

你需要时间排查时

  • 请给我几分钟。
  • 我现在正在检查。
  • 我需要先测试一下。
  • 我先检查线缆和电源。

3. 网络现场对话

3.1 询问用户

  • 你能上网吗？
  • 你能打开这个网站吗？
  • 你能再试一次吗？
  • 你用的是哪个端口？
  • 网络是慢还是完全断了？
  • 其他用户也受影响吗？

3.2 你排障时常说的话

  • 请检查网线。
  • 请重新插一下线。
  • 这个端口掉了。
  • 上联断开了。
  • 交换机离线了。
  • 路由器需要重启。
  • 请 ping 一下网关。
  • IP 地址不对。
  • DNS 设置错了。
  • 无线信号弱。
  • 这条链路有丢包。
  • 网络不稳定。

3.3 你处理完后的表达

  • 我找到问题了。
  • 线松了。
  • 端口之前被禁用了。
  • 我重启了交换机。
  • 我改好了 IP 设置。
  • 网络恢复了。
  • 现在请测试一下。
  • 你现在能访问系统了吗？

3.4 小对话

对话 1：用户上不了网

User:

用户：我上不了网。

You:

你：你给我看一下网线。

You:

你：这根线松了，我重新插一下。

You:

你：你再试一下。

User:

用户：现在好了。

对话 2：网络很慢

User:

用户：网络很慢。

You:

你：所有用户都受影响吗？

User:

用户：是的。

You:

你：我去检查上联和网关。

You:

你：这条线路有丢包。

4. 桌面支持现场对话

4.1 常用问句

  • 你能登录吗？
  • 你看到什么报错？
  • 你能重启电脑吗？
  • 屏幕是黑的吗？
  • 你能听到声音吗？
  • 键盘能用吗？
  • 鼠标能动吗？

4.2 常用说明

  • 电脑无法启动。
  • 系统很慢。
  • 屏幕无信号。
  • 用户忘记密码了。
  • 打印机驱动没装。
  • 你没有权限。
  • 请退出后重新登录。
  • 请重新安装软件。
  • 请更新系统。

4.3 处理结果表达

  • 我重启了电脑。
  • 我重装了驱动。
  • 我重置了密码。
  • 我更新了软件。
  • 打印机现在在线了。
  • 问题已经修复。

4.4 小对话

对话 1：电脑无法登录

User:

用户：我登不上去。

You:

你：请再输一次密码。

User:

用户：还是不行。

You:

你：我给你重置密码。

You:

你：现在再试一下。

对话 2：打印机不能用

User:

用户：打印机离线了。

You:

你：我检查一下线和 IP 地址。

You:

你：驱动坏了，我重装一下。

You:

你：请打印一张测试页。

5. 综合布线现场对话

5.1 开工前确认

  • 安装点在哪里？
  • 需要多少个网络点位？
  • 机柜间在哪里？
  • 线可以走天花上方吗？
  • 两端都需要标签吗？

5.2 施工中常说的话

  • 请先把这些东西移开。
  • 我们要在这里打孔。
  • 我们会把线穿进线管。
  • 走线路径被堵住了。
  • 这根线管满了。
  • 我们需要更长的线。
  • 请确认点位位置。
  • 我们会把设备装到墙上。
  • 请在线两端贴标签。
  • 安装后我们会测试每个端口。

5.3 完工确认

  • 安装完成了。
  • 所有端口都已经测过了。
  • 请检查标签。
  • 请确认点位位置。
  • 验收后请在这里签字。

5.4 小对话

对话 1：确认点位

You:

你：这个办公室需要几个点位？

User:

用户：需要 6 个网络点位。

You:

你：要放在办公桌附近吗？

User:

用户：是的。

对话 2：施工受阻

You:

你：天花上面的走线路径堵住了。

User:

用户：你能换条路线吗？

You:

你：可以，但我们需要更长的线。

6. 监控系统现场对话

6.1 常用问句

  • 哪个摄像头有问题？
  • 是没有画面还是画面不好？
  • 录像机上显示离线吗？
  • 它是什么时候离线的？

6.2 排障表达

  • 摄像头离线。
  • 这个摄像头没有图像。
  • 画面模糊。
  • 请检查供电。
  • 请检查网线。
  • PoE 端口没有链路。
  • 录像机识别不到这台摄像头。
  • 镜头脏了。
  • 红外灯亮着。
  • 我先换线试试。

6.3 处理结果表达

  • 摄像头恢复在线了。
  • 电源适配器故障了。
  • 线坏了。
  • 画面现在正常了。
  • 请再观察一会儿。

6.4 小对话

User:

用户：5 号摄像头离线了。

You:

你：我去检查交换机端口和供电。

You:

你：线坏了，我换一根。

You:

你：摄像头现在恢复在线了。

7. 门禁系统现场对话

7.1 常用问句

  • 哪个门有问题？
  • 读卡器有反应吗？
  • 从里面能开门吗？
  • 门锁有电吗？

7.2 排障表达

  • 读卡器不能用。
  • 门锁没电。
  • 门打不开。
  • 门禁系统离线。
  • 请检查供电。
  • 请检查网络连接。
  • 控制器需要重启。

7.3 完成确认

  • 读卡器现在好了。
  • 门锁正常了。
  • 请再刷一次卡。
  • 问题已经解决了。

8. 打电话或远程支持时常用句

  • 你能听清楚吗？
  • 请告诉我你屏幕上看到了什么。
  • 请点这个图标。
  • 请打开设置页面。
  • 请重启你的电脑。
  • 请发我一张照片。
  • 请把报错信息发给我。
  • 我将远程连接。
  • 请不要关闭这个窗口。
  • 如果问题再出现，给我打电话。

9. 工单、微信群、邮件里好用的简短英文

这些句子非常适合写工单备注。

故障描述

  • 用户无法访问网络。
  • 打印机离线。
  • 摄像头无图像。
  • 电脑无法启动。
  • 门锁没电。

处理过程

  • 已检查电源和线缆。
  • 已重启设备。
  • 已更换网线。
  • 已重装驱动。
  • 已更新 IP 配置。

处理结果

  • 问题已修复。
  • 现在运行正常。
  • 还需要继续观察。
  • 等待用户确认。
  • 需要更换设备。

10. 最常见追问句

现场经常需要继续问，下面这些句子很好用：

  • 它在哪个房间？
  • 它在哪一层？
  • 哪台设备有问题？
  • 影响了多少用户？
  • 这是第一次出现吗？
  • 出问题前你改过什么吗？
  • 你能带我看一下具体位置吗？
  • 你能再测一次吗？
