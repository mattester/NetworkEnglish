# 弱电工程师英语高频背诵版

适合用法：每天背 10 到 20 分钟  
目标：先会认、会读、会说最常用的英文  
原则：不求一次背完，只求反复见面

---

## 1. 使用方法

每天这样背：

1. 先看英文，猜中文。
2. 再看中文，回忆英文。
3. 每个词读 3 遍。
4. 每组挑 5 个词造句。
5. 晚上快速复习一次。

建议顺序：

1. 先背高频 100 词
2. 再背高频句子
3. 再背场景短语

## 5. 最常用 100 句

## 5.1 网络

- `The network is down.` 网络中断了。
- `The internet is slow.` 网络很慢。
- `Please check the cable.` 请检查线缆。
- `Please check the port.` 请检查端口。
- `This port is down.` 这个端口掉了。
- `The link light is off.` 链路灯没亮。
- `Please ping the gateway.` 请 ping 网关。
- `The IP address is wrong.` IP 地址不对。
- `The DNS setting is incorrect.` DNS 设置不对。
- `The switch is offline.` 交换机离线了。
- `The router needs to be restarted.` 路由器需要重启。
- `The signal is weak.` 信号弱。
- `The Wi-Fi is unstable.` 无线不稳定。
- `The uplink is disconnected.` 上联断开了。
- `Please save the configuration.` 请保存配置。
- `Please back up the configuration.` 请备份配置。
- `The firewall may block this traffic.` 防火墙可能拦截了这个流量。
- `There is packet loss on this link.` 这条链路有丢包。
- `The latency is too high.` 延迟太高。
- `Please test again.` 请再测一次。

## 5.2 桌面

- `The computer cannot boot.` 电脑无法启动。
- `The screen is black.` 屏幕黑了。
- `The mouse is not responding.` 鼠标没反应。
- `The keyboard is not working.` 键盘不能用。
- `Please restart the computer.` 请重启电脑。
- `Please log in again.` 请重新登录。
- `The password is incorrect.` 密码错误。
- `The user forgot the password.` 用户忘记密码了。
- `The printer is offline.` 打印机离线。
- `The printer driver is missing.` 打印机驱动没装。
- `Please reinstall the driver.` 请重装驱动。
- `You do not have permission.` 你没有权限。
- `Open the shared folder.` 打开共享文件夹。
- `Please update Windows.` 请更新 Windows。
- `The application cannot open.` 这个程序打不开。
- `Try another browser.` 换个浏览器试试。
- `Please send me the file.` 请把文件发给我。
- `The system is very slow.` 系统很卡。
- `The PC has a blue screen.` 电脑蓝屏了。
- `It is working now.` 现在已经好了。

## 5.3 布线与现场

- `Please label both ends of the cable.` 请在线两端贴标签。
- `Run the cable above the ceiling.` 线从天花上走。
- `Pull the cable through the conduit.` 把线穿过线管。
- `Install the faceplate here.` 在这里安装面板。
- `Terminate the cable on the patch panel.` 把线打到配线架上。
- `Please test all outlets.` 请测试所有点位。
- `The cable route is blocked.` 走线路径堵住了。
- `The conduit is full.` 线管满了。
- `This cable is too short.` 这根线太短了。
- `This cable is too long.` 这根线太长了。
- `Please mount the device on the wall.` 请把设备装在墙上。
- `The rack is full.` 机柜满了。
- `Please check the grounding.` 请检查接地。
- `The label is missing.` 标签缺失。
- `The connector is loose.` 接头松了。
- `Use the tester to check it.` 用测试仪测一下。

## 5.4 监控门禁

- `The camera is offline.` 摄像头离线。
- `There is no image.` 没有图像。
- `The image is blurry.` 图像模糊。
- `Please check the power supply.` 请检查供电。
- `Please check the network cable.` 请检查网线。
- `Restart the NVR.` 重启录像机。
- `The recorder cannot find the camera.` 录像机找不到摄像头。
- `The infrared light is on.` 红外灯亮着。
- `The card reader is not working.` 读卡器不能用。
- `The door lock has no power.` 门锁没电。
- `The door does not open.` 门打不开。
- `The alarm was triggered.` 报警被触发了。

## 5.5 通用表达

- `I am checking it now.` 我正在检查。
- `I found the issue.` 我找到问题了。
- `I replaced the cable.` 我换了线。
- `I restarted the device.` 我重启了设备。
- `Please try again.` 请再试一次。
- `Can you see it now?` 你现在能看到了吗？
- `Can you log in now?` 你现在能登录了吗？
- `The issue has been fixed.` 问题已修复。
- `Please confirm.` 请确认。
- `Thank you for your support.` 感谢配合。

## 7. 三轮背诵法

### 第一轮：认词

只要做到：

- 看到英文知道中文大概意思
- 看到设备能想到英文

### 第二轮：说句子

把词放进句子里：

- `Check the cable.`
- `Restart the switch.`
- `The camera is offline.`

### 第三轮：套场景

比如用户说网络不通，你脑子里直接出现：

- `network`
- `cable`
- `port`
- `gateway`
- `ping`
- `switch`

## 8. 最适合零基础的背法

不要死记字母，按“画面”记。

例如：

- 看到网线，就默念 `cable`
- 看到交换机，就默念 `switch`
- 看到摄像头，就默念 `camera`
- 看到打印机，就默念 `printer`
- 看到机柜，就默念 `rack`

这种方法最适合现场工程师。
